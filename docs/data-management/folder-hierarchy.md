# Folder Hierarchy

## Recommended lab-level hierarchy

```text
LAB_ROOT/
├── admin/
├── shared-resources/
├── templates/
├── projects/
│   ├── project-01-name/
│   ├── project-02-name/
│   └── ...
└── archive/
```

## Recommended project-level hierarchy

```text
project-name/
├── 00_admin/
├── 01_data_raw/
├── 02_data_interim/
├── 03_data_processed/
├── 04_annotations/
├── 05_code/
├── 06_models/
├── 07_experiments/
├── 08_results/
├── 09_reports/
└── 10_archive/
```

## Why numbered folders can help

Numbered folders keep project directories ordered consistently across operating systems and file explorers.
