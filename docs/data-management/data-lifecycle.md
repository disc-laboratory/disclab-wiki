# Data Lifecycle

## Data states

### Raw
Original unmodified data as acquired from sensors, field campaigns, collaborators, or public sources.

### Interim
Temporary outputs produced during cleaning, conversion, or structuring.

### Processed
Curated data ready for modeling, analysis, or publication.

### Results
Outputs generated by analysis, training, reconstruction, or visualization pipelines.

## Rules

- raw data is read-only after ingestion
- processed data must be linked to a script or workflow that produced it
- results should include references to the exact input data and code version
