# Reproducibility Checklist

Use this checklist before sharing results internally or externally.

- [ ] Dataset version is identified
- [ ] Train/validation/test split is fixed and documented
- [ ] Configuration file is saved
- [ ] Random seed is recorded
- [ ] Commit hash is recorded
- [ ] Evaluation script is version controlled
- [ ] Output folder structure is clear
- [ ] Main plots and tables can be regenerated
- [ ] Hardware and runtime constraints are documented
- [ ] Known limitations are stated
