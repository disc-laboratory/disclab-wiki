# Training Workflows

## Minimum training workflow

1. define task and metrics
2. document dataset version and split
3. select baseline model
4. define configuration file
5. run controlled training
6. evaluate on held-out data
7. store checkpoints and logs
8. summarize findings

## Recommendations

- start with a reproducible baseline
- change one major factor at a time
- record failed runs when informative
- separate exploratory work from final benchmark runs
