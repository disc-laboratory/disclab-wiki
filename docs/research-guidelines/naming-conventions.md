# Naming Conventions

## Files and folders

Use lowercase names with hyphens or underscores. Avoid spaces.

Examples:
- `bridge-crack-segmentation`
- `dataset_card.md`
- `exp_20260330_yolo11s_1024.yaml`

## Experiments

Include model, data version, and key setting.

Example:
```text
exp_20260330_yolo11s_dacl10k_v2_img1024
```

## Figures and tables

Use names that reveal content.

Examples:
- `fig_bridge_crack_qualitative_examples.png`
- `tab_ablation_resolution_vs_map.csv`
