# NeRF and Gaussian Splatting

## Intended use

These methods are useful for novel view synthesis, scene representation, and visualization-oriented digitalization workflows.

## Minimum reporting items

- input image source and count
- camera pose source
- model type and implementation
- training settings
- rendering outputs generated
- limitations for metric or geometric use

## Important note

State clearly when an output is intended for visualization versus geometry extraction or engineering analysis.
