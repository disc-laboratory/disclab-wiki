# LOD Definitions

## Purpose

LOD definitions should be adapted to the engineering use case.

## Recommended distinctions

- **LOD for visualization**: adequate for communication and XR review
- **LOD for documentation**: suitable for traceable records of assets and damage
- **LOD for analysis**: sufficiently structured for downstream mechanical or engineering interpretation

## Minimum requirement

Every project using the term LOD must define explicitly what geometric, semantic, and condition information is included.
