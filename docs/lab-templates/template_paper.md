# DISCLab Paper Template

This page provides the official academic paper template used in the DISC Laboratory.

## Description

The paper template is designed for writing academic manuscripts, including both full papers and abstract-only submissions. It follows a clean and conservative structure suitable for:

- journal papers;
- conference papers;
- extended abstracts;
- technical reports.

The template supports both abstract-only and full-paper modes and is compatible with standard academic submission formats.

## Files

- [Download the LaTeX paper template](files/DISCLab_paper_template.tex)
- [Download the references file](files/references.bib)
- [Download the PDF example](files/DISCLab_paper_template.pdf)

## Usage

- Use this template as a starting point for writing papers;
- Select the appropriate mode (abstract-only or full paper) using the toggle in the `.tex` file;
- Adapt sections and structure according to the target venue requirements;
- Ensure formatting remains clean and consistent.

## Notes

- Ensure all figures are high quality and readable;
- Avoid overcrowded slides — prioritize clarity;
- Maintain consistent formatting across slides;
- Use the template to reinforce a coherent lab identity.
- This template requires a bibliography file named `references.bib`;
- Ensure all referenced images, logos, or folders used in the `.tex` file are available locally before compiling.
- Compile using a LaTeX workflow compatible with bibliography processing.
- No predefined image folder is given. Organize figures in a dedicated directory (e.g., `figures/`);
