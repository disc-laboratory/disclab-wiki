# DISCLab CV Template

This page provides the official CV template used in the DISC Laboratory.

## Description

The CV template is designed to present academic and professional information in a clear, structured, and professional format. It is suitable for:

- academic CVs;
- job and fellowship applications;
- grant applications;
- professional profiles.

The template follows a clean, single-column layout with a conservative and readable style aligned with academic standards.

## Usage

- Use this template as a starting point for your CV;
- Create a copy before editing;
- Adapt sections to your profile and career stage;
- Keep formatting consistent throughout the document.

## Notes

- Ensure all figures are high quality and readable;
- Avoid overcrowded slides — prioritize clarity;
- Maintain consistent formatting across slides;
- Use the template to reinforce a coherent lab identity.
- This template requires a bibliography file named `publications.bib` for listing publications;
