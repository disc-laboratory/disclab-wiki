# DISCLab Slides Template

This page provides the official slide template used in the DISC Laboratory for presentations.

## Description

The slides template is designed to ensure a consistent and professional visual identity across all presentations from the lab. It is suitable for:

- conference presentations;
- internal group meetings;
- project updates;
- lectures and seminars.

The template follows a clean structure aligned with NTU and DISC Lab branding.

## Files

- [Download the LaTeX slides template](files/DISCLab_slides_template.tex)
- [Download the references file](files/refs.bib)
- [Download the PDF example](files/DISCLab_slides_template.pdf)

## Usage

- Use this template as a starting point for all presentations;
- Create a copy before editing;
- Keep the overall structure and style consistent;
- Adapt content (figures, sections, examples) to your specific presentation.

## Notes

- Ensure all figures are high quality and readable;
- Avoid overcrowded slides — prioritize clarity;
- Maintain consistent formatting across slides;
- Use the template to reinforce a coherent lab identity.
- This template also requires an associated `.bib` file (`refs.bib`) for references.
- Ensure all referenced images, logos, or folders used in the `.tex` file are available locally before compiling.
- No predefined image, videos and logos folders are provided. Organize them in a dedicated directories (e.g., `figures/`; `brand\`; `videos\`);
- Compile using a LaTeX workflow compatible with bibliography processing.
