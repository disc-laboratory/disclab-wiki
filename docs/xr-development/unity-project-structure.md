# Unity Project Structure

## Recommended structure

- `Assets/Scripts/`
- `Assets/Scenes/`
- `Assets/Prefabs/`
- `Assets/Materials/`
- `Assets/Models/`
- `Assets/StreamingAssets/`
- `Assets/Plugins/`
- `Assets/Resources/` only when justified

## Rules

- keep scenes minimal and modular
- separate UI, inference, and visualization logic
- document all external plugins and versions
